# Session Planner - Features Guide

## üÜï New Features

### 1. Session Library

**Save Your Sessions**
- Click "Save Session" button in the top right
- Sessions are stored in your browser's localStorage
- Counter shows how many sessions you have saved
- All session details are preserved: title, date, objectives, notes, resources, and agenda

**Load Saved Sessions**
- Click "Session Library" to view all saved sessions
- See session details at a glance
- Click "Load" to open a saved session
- Perfect for reusing lesson plans or meeting templates

**Delete Sessions**
- Click "Delete" on any session in the library
- Confirmation prompt prevents accidental deletion
- Permanently removes session from localStorage

**Session Library Features**
- Visual card layout showing key session info
- Date saved timestamp
- Activity count
- Quick preview of session details

### 2. Export to PDF

**Professional PDF Export**
- Click "Export PDF" button when viewing a session
- Automatically formatted, professional-looking PDF
- Includes all session information:
  - Session title, date, time, participants
  - Total duration calculation
  - Session notes
  - Learning objectives (numbered list)
  - Session resources with links
  - Complete agenda with:
    - Activity names and numbers
    - Start and end times
    - Duration and type
    - Activity-specific notes
    - Activity-specific resources

**PDF Features**
- Automatic page breaks
- Text wrapping for long content
- Professional typography
- Download with session title as filename
- Ready to share or print

### 3. New Session Button

**Start Fresh**
- Click "New Session" to clear all fields
- Confirmation prompt if you have unsaved work
- Quickly start planning a new class or meeting
- Doesn't affect your saved sessions

## üíæ Data Persistence

**What Gets Saved:**
- All sessions you explicitly save
- Custom activities you create
- Stored in browser localStorage

**What Doesn't Get Saved Automatically:**
- Current working session (until you click "Save Session")
- Sessions are only saved when you click the save button

**Important Notes:**
- Data is stored per browser
- Clearing browser data will delete saved sessions
- Export important sessions to PDF for backup
- No cloud sync (everything is local)

## üéØ Common Workflows

### Creating a Reusable Template

1. Plan your session completely
2. Click "Save Session"
3. Name it descriptively (e.g., "Weekly Team Meeting Template")
4. Load it whenever you need that structure
5. Modify dates/times as needed
6. Save as a new session or update the original

### Planning Multiple Similar Classes

1. Create your first class session
2. Save it with a specific name (e.g., "Week 1 - Introduction")
3. Click "New Session"
4. Load the saved session
5. Modify for next class
6. Save with new name (e.g., "Week 2 - Advanced Topics")

### Exporting for Distribution

1. Complete your session planning
2. Click "Export PDF" for professional sharing
3. Email PDF to participants
4. Also save session in library for future reference
5. Click "Export TXT" if you need plain text backup

## üö® Troubleshooting

**Sessions Not Saving?**
- Check if browser is in incognito/private mode
- Verify localStorage is enabled in browser settings
- Check available browser storage space

**PDF Not Downloading?**
- Disable pop-up blocker for this site
- Check browser download settings
- Try a different browser

**Lost My Sessions?**
- Check if you're in the same browser
- Browser data may have been cleared
- Import from PDF exports if available

**Session Library Not Showing Sessions?**
- Refresh the page
- Check browser console for errors
- Verify localStorage isn't full
